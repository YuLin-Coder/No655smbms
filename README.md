## 项目介绍
基于SSM框架的超市管理系统，有用户和管理员两个角色，主要功能如下
销售管理、商品管理、库存管理、供货商管理、员工管理、会员管理、系统管理

## 项目技术
- 编程语言：Java
- 数据库：MySQL
- 前端技术：JSP、JavaScript、bootstrap、JQuery
- 后端技术：Spring、SpringMVC、MyBatis

## 运行环境
- JDK版本：JDK1.8及以上
- 开发工具：IDEA、Ecplise、Myecplise都可以
- 数据库: MySQL5.7及以上
- Maven：maven3.0及以上
